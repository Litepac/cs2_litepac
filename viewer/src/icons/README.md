# CS2 Icon Catalog

This folder is an intentional source-owned catalog of extracted CS2 SVG icons used by replay, utility, equipment, and killfeed surfaces.

Do not treat currently unreferenced icons as dead code. DemoRead expects to reuse more of this catalog as review tools expand, and keeping the set local avoids repeated asset extraction churn.

Cleanup rule:
- Remove an icon only when it is known to be invalid, duplicated, legally unsuitable, or replaced by a better repo-owned equivalent.
- Do not trim icons just because a current component does not import them.
